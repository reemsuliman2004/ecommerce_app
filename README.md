# ecommerce_app
تطبيق Flutter متكامل يقدّم متجرًا إلكترونيًا خاصًا بالمستودعات، حيث يتيح للمستخدمين استعراض المستودعات المتوفّرة، تصفّح التصنيفات،عرض المنتجات، بالإضافة إلى إدارة سلة المشتريات وتتبع الطلبات بطريقة سهلة ومنظّمة. يركّز التطبيق على تقديم تجربة مستخدم سلسة ومظهر حديث، مع دعم كامل للغة العربية،واعتماد بنية برمجية واضحة لسهولة التطوير والتوسّع مستقبلًا.
